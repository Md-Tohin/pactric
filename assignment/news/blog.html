Variables declared with var and const are scoped to the immediaate function body. Variables declared with the var keyword are hoisted. Hoisting means that the variable can be accessed in their enclosing scope even before they are declared. Variables declared with the let keyword are block-scoped, which means the variables will have scope to the immediate enclosing block.

Regular functions created using function declarations or expressions are constructible and callable. Since Regular functions are constructors, they can be called using the new keyword. In regular function, this changes according to the way that function is invoked.

Arrow functions are only callable and not constitutable, i.e. arrow functions can never be used as constructor functions. Arrow functions don't have their own “this”, and they don’t redefine the value of “this ”within the function.

Template strings are a powerful feature of modern JavaScript released in ES6. It lets us insert/interpolate variables and expressions into strings without needing to concatenate like in older versions of JavaScript. It allows us to create strings that are complex and contain dynamic elements.
